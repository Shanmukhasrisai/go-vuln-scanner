# GoVulnScanner
## Overview
GoVulnScanner is a vulnerability scanner written in Go, similar to Nuclei, designed to efficiently detect security vulnerabilities. It offers features like multi-threaded scanning, CI/CD integration, and a professional API framework for seamless security automation. Suitable for security teams and researchers, it provides scalable, production-ready security scanning capabilities.
## Integrated CI/CD Support
GoVulnScanner features seamless CI/CD integration using GitHub Actions. Every code commit and pull request to the main branch triggers automated build, dependency install, security vetting, and testing for fast feedback and reliable deployments.
